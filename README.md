# نظام إدارة العيادات - لوحة Filament

نسخة من نظام العيادات الأصلي (PHP) مبنية بـ **Laravel 11** و **Filament v5** بنفس الوظائف: إدارة العيادات، طباعة التذاكر، والإحصائيات.

## المتطلبات

- PHP 8.2+
- Composer
- قاعدة بيانات (SQLite افتراضي، أو MySQL في `.env`)

## التثبيت والتشغيل

### 1. تثبيت الاعتماديات (إن لم تكن مثبتة)

```bash
cd clinic-dashboard
composer install
```

### 2. إعداد البيئة

```bash
cp .env.example .env
php artisan key:generate
```

إذا أردت استخدام **MySQL** بدل SQLite، عدّل في `.env`:

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=clinic_dashboard
DB_USERNAME=root
DB_PASSWORD=
```

ثم أنشئ قاعدة البيانات وشغّل الهجرات:

```bash
php artisan migrate
```

### 3. إنشاء مستخدم للوحة التحكم

```bash
php artisan make:filament-user
```

أدخل الاسم، البريد الإلكتروني، وكلمة المرور عند الطلب.

### 4. تشغيل المشروع

```bash
php artisan serve
```

ثم افتح المتصفح على: **http://localhost:8000/admin**

## الوظائف (مطابقة للنظام الأصلي)

| الوظيفة | الوصف |
|--------|--------|
| **العيادات** | إضافة، تعديل، حذف العيادات من لوحة Filament |
| **طباعة تذكرة** | من جدول العيادات: زر "طباعة تذكرة" يفتح التذكرة في تاب جديد ويُزيد رقم المريض تلقائياً |
| **التذكرة** | صفحة طباعة تحتوي: اسم العيادة، رقم التذكرة، التاريخ، الوقت، اسم المستشفى |
| **الإحصائيات** | صفحة "الإحصائيات": جدول (اسم العيادة + عدد المرضى)، زر "إعادة تعيين الإحصائيات"، و"طباعة التقرير" |

## المسارات

- `/` → تحويل إلى `/admin`
- `/admin` → لوحة Filament (تسجيل الدخول ثم إدارة العيادات والإحصائيات)
- `/ticket/{clinic}` → عرض وطباعة تذكرة للعيادة (يُزيد رقم المريض تلقائياً)

## الشعار

تم نسخ `assets/logo.jpg` من المشروع الأصلي إلى `public/images/logo.jpg` ويُعرض في صفحة التذكرة إن وُجد.

## ملاحظات

- المشروع الأصلي (PHP) يبقى في المجلد الجذر؛ هذا المشروع في مجلد `clinic-dashboard` فقط.
- لاستيراد بيانات قديمة من قاعدة النظام الأصلي، انسخ جدول `clinics` إلى قاعدة Laravel أو استخدم Seeder.
